# Experiments

## Experiment 1: Prompt Consistency

Goal: Evaluate how consistent model responses are across repeated queries.

Method:
- Same prompt repeated 10–20 times
- Compare variations in outputs

## Experiment 2: Adversarial Inputs

Goal: Test how the model handles ambiguous or conflicting instructions.

Method:
- Create prompts with conflicting constraints
- Analyze which instruction the model prioritizes
